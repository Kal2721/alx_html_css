FRONT END WEB DEVELOPMENT

Front-end web development focuses on the user interface (UI) of a website or application, the part users see and interact with directly. It involves using HTML, CSS, and JavaScript to create interactive and visually appealing designs.

What is HTML?

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

What is HTML used for?

It's used to organize content, including text, images, videos, and other elements, on a website.

What is CSS?

CSS (Cascading Style Sheets) is a language designed to simplify the process of making web pages presentable.
